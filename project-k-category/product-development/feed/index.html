One of the major steps in a student’s life is higher education as we all agree on. Most probably, if someone has succeeded in high school education, he/she is going to apply for a university to build his/her carrier path through higher education. On this matter, this web app helps them out. Nowadays, social media is more popular than other printed and television media. Thus, this app was developed to get a better understanding about the popularity of universities in social media. This app presents how people have been reacted about the university you are interested in; both positively and negatively, which will help you in making a better choice as a university expected student. More than 2000 tweets have been extracted for each university in order to increase the accuracy level of this app. The extracted tweets have been analyzed separately as good and bad and shown in percentages and also graphically. Additionally, this web app shows frequency distribution, popularity variations and few other details.

One of the biggest problems in the University of Peradeniya, there is no standardized system to keep track of all projects and researches. Our knowledge hub provides the facility to upload and retrieve details of a project or research effectively. This system is very useful for upcoming projects and researches to analyze the performance level. This system saves time and keeps track of all the data safe. To develop the platform to get current and previous details of projects and researches. The main objective of our project is to make the use of the system very much easier to access and to analyze the achievement level of all projects and researches. Everyone can access the system. Only the students and academic staffs can upload the data. This system is only for our university. There are two interfaces. They are searching interface and upload interface. The authorized users can upload documents as pdf. Everyone can search the uploaded documents. The technologies that are used for our project are java based Spring Boot to design REST-API for Backend, HTML, CSS, JavaScript for Frontend, MySQL for database, Apache Tika for PDF Extraction and Elasticsearch. This is a simple and efficient search engine. It is user-friendly. This can be accessed immediately. New students gain more knowledge regarding previous projects and researches.

This is a next generation text book for RACGP exam candidates. RACGP exam means Royal Australian College of General Practitioners Fellowship exam. RACGP Fellowship exams access a candidate’s competency for unsupervised general practice anywhere in Australia. Our project owner, Dr. Ramindhu Galgamuwa (FRACGP Phd) will be doing a series of webinars for Australian doctors sitting the specialist exams. The web that we created would be where they access the notes – and write notes. Admin/Project owner will add notes under each topic through this web application. The notes are accessible only for paid members. Therefore this application includes a payment gateway. While referring the notes the users can highlight any important thing. Not only that but also they can add their own notes in your notes page. There is an option which can be used to manage all the users of this web application. Any RACGP exam candidate can get more information about this web application; how to add payments, how to login to the application, how to access the notes and also how to download notes. Home page contains a index of topics which are added by the admin and they can decide whether it is useful or not. As mentioned above this is a user friendly web application.

Our project aims to create a database of Research papers, Project Reports, Thesis’ done in University of Peradeniya. Since there is no current database where we keep the project reports other than the hard copies themselves, it is not possible for a person outside the university to access these reports. We created a platform where anyone in the university can upload their respective reports into a database which can be used by anyone; inside or outside the university, to access and read through. Anyone who is a member of the University of Peradeniya can use the CMS platform we have created to register on Pera Knowledge Portal. Members of our website can upload a pdf document of the documentation (Research Paper, Project Report, Thesis) they wish to be published. Knowledge Portal platform extracts the texts from pdf documents and indexed them in our database. Which gives the documentation to be searched even by the content of pdf documents. Anyone inside or outside the university can use the search engine we have provided to search these documents. Pera knowledge Portal search engine is able to produce accurate search results and give the most relatable documents according to the search text by the user.
